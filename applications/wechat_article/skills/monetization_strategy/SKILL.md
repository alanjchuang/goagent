# 公众号内容变现策略

当任务涉及公众号文章创作、选题、发布或增长时，优先使用以下变现框架。

## 1. 变现路径选择

根据用户输入选择主路径，不要在一篇文章里堆叠过多目标：

- 私域引流：关注公众号、回复关键词、查看历史合集、加企微/入群、咨询预约；资料包领取仅作为可选项，不要默认强制使用。
- 知识产品：课程、训练营、陪跑、模板包。
- CPS/联盟：工具测评、书单、课程推荐、软件订阅。
- 广告合作：品牌露出、软文、栏目冠名。
- 服务转化：顾问服务、企业内训、定制方案。

## 2. 文章结构与转化匹配

- 痛点型文章：适合咨询、训练营、社群转化。
- 清单型文章：适合关注/回复关键词、查看合集、CPS、工具包转化；资料包转化为可选方案。
- 测评型文章：适合 CPS、品牌合作、工具推荐。
- 案例型文章：适合课程、陪跑、咨询转化。
- 教程型文章：适合模板、工具、社群转化。

## 3. CTA 写法

CTA 要自然、具体、低压，不要诱导分享或承诺收益。

推荐写法：

- “如果你想继续看这类实测，可以关注后回复「关键词」，我会把后续更新放在同一合集里。”
- “如果你正在做类似尝试，可以在评论区说说你的场景，我会优先补充对应案例。”
- “如果你正在做类似尝试，可以先用文末行动清单自查一遍。”
- “需要模板/资料的读者，可以根据文末说明获取可编辑版本。”

避免写法：

- “保证你月入过万。”
- “不转发就亏了。”
- “稳赚不赔。”
- “零成本躺赚。”

## 4. 输出要求

每篇文章至少输出：

1. 主转化目标。
2. CTA 位置和文案。
3. 读者下一步动作。
4. 风险说明或免责声明。
5. 可复用到朋友圈/社群/小红书的分发文案。
